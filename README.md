# DarkestDungeon-archive-helper

暗黑地牢自动存档/回档助手（Windows）。

## 运行环境

- Python 3.10+
- Java 运行环境（用于 `tools/DDSaveEditor.jar`）
- Windows（当前热键与进程检测逻辑基于 Windows）

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动 GUI

```bash
python dd_helper.py gui
```

## 其他命令

```bash
python dd_helper.py verify
python dd_helper.py monitor
```
